# Análisis de Plantaciones

Herramientas utilizadas:
- Excel
- SQL / MySQL
- Power BI

Descripción:
Proyecto de análisis de datos enfocado en el estudio de
plantaciones agrícolas, evaluando distribución, volumen
y tendencias mediante indicadores y dashboards para
apoyar la toma de decisiones productivas.
